開新專案

當安裝好Eclipse + PyDev我們就可以開始使用它來開發專案，首先先選擇File -> New -> Project

會跳出一個新專案類形選擇視窗，選擇Pydev Project後按Next

之後會出現讓你設定專案相關設定的視窗，輸入你專案的名稱，選擇你所使用的Python版本後按Finish，就完成了開新專案

開新模組

光有專案是無法執行的，我們接著介紹如何開新模組，選擇File -> New -> Pydev Module

接著會有開新模組的視窗跑出來，我們在此設定模組的名稱為hello後按Finish，注意名稱不用加.py，它會自動幫我們加

接著我們就有了第一個python的模組 : hello.py

執行模組

我們在hello.py中輸入一些程式碼後，就可以開始執行，在上方的工具列上面找到執行的按鈕

在這之後會出現一個讓你選擇執行方式的視窗，通常我們選澤Python Run

然後hello.py就開始執行，結果會出現在下面的Console視窗中，到此就可以開始使用PyDev開發Python專案
